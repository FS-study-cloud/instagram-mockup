# 인스타그램 목업 만들기

## 개요

인스타그램 화면을 목업으로 구성하여 HTML, CSS 그리고 JS에 대해 복습해보기 위해 진행되었다.

## 기대 효과

- HTML 마크업 기술 향상
- CSS 스타일링 및 반응형 웹 구현 기술 향상
- UI/UX 디자인 이해
- 협업 능력

## 기능적 요구사항

1. 다크모드
    1. 달 버튼을 눌렀을 때 다크모드가 활성화된다.
2. 미디어쿼리/반응형 화면
    1. 화면의 크기에 따라 화면 구성이 달라진다.
    2. 웹과 앱을 나누어 만들지 않도록 한다.

## 회고

- 미숙하지만 최선을 다했다.
- 이후 다른 미션들, 프로젝트를 위해 깃허브를 사용해보았다.
- 팀이 결성된 후 첫 미션이었고 기한을 정해두지 않아 시간이 많이 소요되었다.
- 수업 영상 안의 실습에서 나온 인스타그램 화면으로 진행해 실제 인스타그램 화면과 차이가 있다.
- 무한 스크롤을 이용해 실제 인스타그램처럼 만들고싶다.
- 스토리 버튼을 누르면 새로운 화면이(스토리) 뜨도록 하고싶다.
